Digesting Care
 The Digesting Care publication was developed for the EU commission project Taking Care by Museum Volkenkunde in the Netherlands. It is a static website written by a python script that takes the content written in markdown and make it HTML using Pandoc. The website interface filters the content based on a question generated by a small set of parameters. These are defined by us digesting the concepts and unsolved topics of the conference. The website was developed experimenting with the idea of resilient web design and web-to-print techniques. We printed an edition of 120 copies of 50 different publications.
